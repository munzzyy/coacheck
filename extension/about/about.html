coacheck

Reads a Certificate of Analysis (COA) - the lab report that ships with a research peptide vial - straight off your screen, and computes the purity/reconstitution math and a red-flag checklist on it. Nothing here judges whether to use anything; it does the math on what you give it.

What it reads, and what it doesn't

Only the on-screen rectangle you drag, on the one tab you're on, when you ask for it - nothing runs on any page until you click the toolbar icon or press the keyboard shortcut. OCR (Tesseract.js, bundled with this extension) runs entirely on your machine. There's no server component, no analytics, and no network request this extension ever makes - every file it uses (the OCR engine, the language model, the code that parses COA fields) ships inside the extension package.

Permissions, and why

  • activeTab - lets the toolbar click or keyboard shortcut capture a screenshot of the current tab, for the one tab and the one moment you invoked it. Not a standing grant.
  • scripting - injects the region-select box and the results panel into the page you're on, only when you trigger it.
  • offscreen (Chrome only) - Chrome's service worker background has no document, so OCR runs in a hidden offscreen page instead. Firefox doesn't need this; its background page already has one.

No host_permissions, no tabs, no storage. This extension keeps no settings and remembers nothing between uses.

Informational tool only - not medical advice, and it does not endorse, recommend, or source any compound. It checks a document for missing or implausible data; it can't tell you whether the document itself is genuine.
